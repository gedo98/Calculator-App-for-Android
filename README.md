# Smart Calculator 
1.الوصف (Description)
هذا المشروع عبارة عن آلة حاسبة برمجية بسيطة تقوم بإجراء العمليات الحسابية الأساسية (الجمع، الطرح، الضرب، والقسمة) بدقة وسهولة.

2.تعليمات التثبيت (Installation)
لتشغيل المشروع على جهازك:
1. قم بتنزيل ملفات الكود (Code) من هذا المستودع.
2. تأكد من تثبيت بيئة التشغيل المناسبة (مثل Python أو مجرد متصفح إذا كان ويب).
3. شغل الملف الرئيسي للمشروع.

3. مثال الاستخدام (Usage Example)
بعد تشغيل البرنامج، يمكنك إدخال الأرقام كالتالي:
- الرقم الأول: 10
- العملية: +
- الرقم الثاني: 5
- النتيجة ستكون: 15
